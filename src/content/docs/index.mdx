<Aside type="note">Oicana is in Alpha. It is rough around the edges and has a limited number of integrations.</Aside>

<CardGrid stagger>
	<Card title="Powerful Layouting" icon="open-book">
		Builds on the <a href="https://typst.app/">modern open source typesetter Typst</a>. Oicana templates can use all of Typst's functionality including its <a href="https://typst.app/universe/">extensive package ecosystem</a>.
	</Card>
	<Card title="Performant" icon="rocket">
	  Templates can be "warmed up" and can produce non trivial PDF files in single digit milliseconds.
	</Card>
	<Card title="Multi-platform" icon="seti:db">
		The same templates can be compiled from the browser, a C# application or a Node.js backend. Integrations for more platforms are in development.
	</Card>
	<Card title="Version Control Ready" icon="seti:git">
	  Templates are plain text files and can be checked in to version control.
	</Card>
	<Card title="Escape Vendor Lock-in" icon="seti:license">
    Reuse templates with other Typst based solutions; The Typst compiler is open source!
	</Card>
</CardGrid>
